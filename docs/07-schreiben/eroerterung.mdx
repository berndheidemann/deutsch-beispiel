# Erörterung

## Erörterungsformen

| Form | Aufbau | Einsatz |
|------|--------|---------|
| **Lineare Erörterung** | These → Argument 1 → Argument 2 → Argument 3 → Fazit | Wenn die Position klar ist |
| **Dialektische Erörterung** | Pro-Argumente → Contra-Argumente → Synthese/Fazit | Wenn beide Seiten beleuchtet werden sollen |

## Argumentationstypen

| Typ | Beschreibung |
|-----|-------------|
| **Faktenargument** | Stützt sich auf nachprüfbare Tatsachen |
| **Autoritätsargument** | Beruft sich auf Experten/Autoritäten |
| **Normatives Argument** | Beruft sich auf Werte und Normen |
| **Analogie-Argument** | Vergleich mit ähnlichem Sachverhalt |
