# Metrum & Reimschema

## Versfüße

| Versfuß | Schema | Beispiel | Wirkung |
|---------|--------|---------|---------|
| **Jambus** | ◡ ▬ | ge-DICHT | Fließend, natürlich |
| **Trochäus** | ▬ ◡ | MÜL-ler | Bestimmt, feierlich |
| **Daktylus** | ▬ ◡ ◡ | WAN-de-rer | Tänzerisch, beschwingt |
| **Anapäst** | ◡ ◡ ▬ | Pa-ra-DIES | Drängend, vorwärtstreibend |

## Reimschemata

| Schema | Muster | Beispiel |
|--------|--------|---------|
| **Paarreim** | aabb | Volkslied, Ballade |
| **Kreuzreim** | abab | Häufigste Form |
| **Umarmender Reim** | abba | Rahmend, geschlossen |
| **Schweifreim** | aabccb | Sechszeilige Strophe |

## Übungen

### Metrum erkennen

So sieht ein Jambus aus:

<MetrumVisualizer
  vers={metrumVisualizerData[0].vers}
  silben={metrumVisualizerData[0].silben}
  versfuss={metrumVisualizerData[0].versfuss}
  kadenz={metrumVisualizerData[0].kadenz}
/>
