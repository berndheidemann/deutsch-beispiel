# Textanalyse Lyrik

In diesem Kapitel lernst du die Methoden der **Lyrik-Analyse**: Metrum, Reimschema, Bildsprache und die systematische Gedichtanalyse.

## Lernziele

- Versfüße (Jambus, Trochäus, Daktylus, Anapäst) bestimmen
- Reimschemata erkennen und benennen
- Bildsprache in Gedichten analysieren
- Eine vollständige Gedichtanalyse durchführen

:::warning Wichtig
Das **lyrische Ich** ist **nicht** der Autor! Formulierung: „Das lyrische Ich beschreibt…", NICHT „Goethe beschreibt…"
:::
