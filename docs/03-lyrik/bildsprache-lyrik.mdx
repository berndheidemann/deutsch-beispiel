# Bildsprache in der Lyrik

Bildliche Sprache hat in der Lyrik eine besondere Bedeutung, da Gedichte auf engem Raum eine hohe Bedeutungsdichte erreichen.

## Beispiel: Eichendorff — „Mondnacht"

> „Es war, als hätt der Himmel / Die Erde still geküsst, / Dass sie im Blütenschimmer / Von ihm nun träumen müsst."
>
> — Joseph von Eichendorff, *Mondnacht* (1837)

### Bildsprache-Analyse

- **„der Himmel die Erde geküsst"** — Personifikation: Himmel und Erde werden als handelnde Wesen dargestellt, die Verbindung von Himmel und Erde symbolisiert die romantische Sehnsucht nach Einheit
- **„Blütenschimmer"** — Metapher: Das Kompositum verbindet Blüte und Licht zu einem synästhetischen Bild
- **„träumen müsst"** — Personifikation: Die Erde wird als träumendes Wesen dargestellt
