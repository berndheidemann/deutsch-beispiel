# Textanalyse Epik

In diesem Kapitel lernst du die Methoden der **epischen Textanalyse**: Erzähltechnik, Figurenanalyse und Zeitgestaltung.

## Lernziele

- Erzählperspektiven nach Stanzel bestimmen und ihre Wirkung analysieren
- Erlebte Rede sicher erkennen und von anderen Darbietungsformen unterscheiden
- Zeitgestaltung (Raffung, Dehnung, Deckung) analysieren
- Figuren systematisch charakterisieren

## Textbasis

Unsere Leitwerke in diesem Kapitel:
- **Theodor Fontane** — *Effi Briest* (Poetischer Realismus)
- **Franz Kafka** — *Die Verwandlung* (Expressionismus/Moderne)
- **E.T.A. Hoffmann** — *Der Sandmann* (Romantik)
