# Dramentheorie

## Geschlossene vs. offene Dramenform

| Merkmal | Geschlossenes Drama | Offene Dramenform |
|---------|-------------------|-----------------|
| **Aufbau** | 5-Akt-Schema (Freytags Pyramide) | Szenenfolge ohne klassischen Spannungsbogen |
| **Handlung** | Einheit von Ort, Zeit, Handlung | Episodisch, fragmentarisch |
| **Figuren** | Individuelle Charaktere, oft adelig | Typen, soziale Rollen, niedere Stände |
| **Sprache** | Vers, gehobene Sprache | Alltagssprache, Dialekt |
| **Ziel** | Katharsis | Gesellschaftskritik |
| **Beispiel** | Schiller: *Kabale und Liebe* | Büchner: *Woyzeck* |

## Freytags Pyramide

1. **Exposition** — Einführung in Ort, Zeit, Figuren
2. **Steigende Handlung** — Konfliktverschärfung
3. **Klimax / Peripetie** — Höhe- und Wendepunkt
4. **Fallende Handlung** — Retardierendes Moment
5. **Katastrophe / Lösung** — Tragischer Ausgang oder Auflösung
