# Dialoganalyse

Die Dialoganalyse ist das zentrale Werkzeug der Drameninterpretation.

## Methodik

1. **Wer** spricht? (Figur, Status, Beziehung)
2. **Wie** wird gesprochen? (Sprachebene, Stilmittel, Regieanweisungen)
3. **Was** wird gesagt — und was nicht? (Subtext)
4. **Warum** wird so gesprochen? (Intention, Machtdynamik)

:::tip Tipp
Dramen-Dialoge lassen sich hervorragend mit **Kommunikationsmodellen** analysieren — z.B. mit dem 4-Ohren-Modell von Schulz von Thun (→ Kapitel 6).
:::
